# cgroups + MIG

### File Descriptions

- `mig_flags.sh`: A bash script to enable or disable MIG CPU affinity settings as well as offering other toggles via flags

- `mig_easy_setup.sh`: A bash script that automates the setup of MIG CPU affinity settings for 7 partitions of equal size (maximum)

- `setup_mig_cpu_affinity.sh`: A bash script that sets up MIG CPU affinity settings based on user-defined partition sizes. This script has some errors related to permissions for cgroups control and management.
